# Dose Calculation Plugin (WordPress)

A lightweight and efficient WordPress plugin designed to calculate medical dosage values based on user inputs.  
Built using PHP and JavaScript, the plugin provides real-time calculations, validation, and a clean UI to enhance accuracy and reduce manual errors for medical or clinical workflows.
